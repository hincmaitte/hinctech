# HincTech

HincTech es un proyecto desarrollado como parte de un proceso de aprendizaje en desarrollo web, enfocado en la práctica y experimentación con tecnologías web modernas y herramientas de inteligencia artificial.

El sitio presenta información sobre servicios de reparación de **celulares, notebooks y PCs**, incluyendo detalles sobre tiempos estimados de trabajo, condiciones de garantía y medios de contacto. La estructura de la página está pensada para ser clara, simple y funcional, facilitando que cualquier usuario pueda encontrar rápidamente la información que necesita.

Este proyecto forma parte del portafolio personal de Maitte Hinc y refleja su proceso de aprendizaje en desarrollo web asistido por inteligencia artificial. Durante el desarrollo se utilizaron herramientas de IA como ChatGPT y Codex para asistir en la generación de código, la organización de la estructura del sitio y la implementación de distintas funcionalidades.

El objetivo de **HincTech** es servir tanto como práctica de desarrollo web como ejemplo de cómo las herramientas de inteligencia artificial pueden integrarse en el flujo de trabajo de programación para construir proyectos reales mientras se adquieren nuevas habilidades.
