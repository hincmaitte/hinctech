HincTech es un proyecto de aprendizaje en desarrollo web, creado para practicar y experimentar con tecnologías web y herramientas de inteligencia artificial. La página presenta servicios de reparación de celulares, notebooks y PCs, junto con información clara sobre tiempos de trabajo, garantía y contacto.
Este proyecto refleja el proceso de aprendizaje de la autora, quien va creando y publicando cada proyecto mientras adquiere nuevas habilidades en desarrollo web y IA. Por ejemplo, esta página fue desarrollada utilizando ChatGPT y Codex, demostrando cómo la IA puede asistir en la creación de proyectos web reales.
El objetivo de HincTech es servir tanto como portafolio personal como un ejemplo práctico de cómo aplicar herramientas modernas de IA en desarrollo web, mientras se aprende a estructurar páginas funcionales y visualmente atractivas.
